# Простой анализатор логов nginx
Решение домашнего задания №1.

Пример запуска:
  - `python log_analyzer.py`
  - Параметры запуска:
      - `-h` для справки
      - `-c <путь к конфигурационному файлу>`
      - `-l <путь для записи логов скрипта>`

Запуск тестов:
  - `python log_analyzer_tests.py -v`
